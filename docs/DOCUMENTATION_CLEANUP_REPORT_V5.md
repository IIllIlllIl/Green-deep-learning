## 命名规范统一

### 已应用的规范
- **核心文档**: 大写+下划线 (如`QUICK_REFERENCE.md`)
- **详细文档**: 小写+下划线 (如`performance_metrics_analysis.md`)
- **总览文档**: `FEATURES_OVERVIEW.md`
- **快速参考**: `*_QUICKREF.md`

### 文档优先级标记
- ⭐⭐⭐ (3星) - 必读文档
- ⭐⭐ (2星) - 推荐阅读
- ⭐ (1星) - 按需参考
