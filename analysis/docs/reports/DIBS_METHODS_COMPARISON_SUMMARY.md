## 方法选择指南

### 选择全局标准化方法的场景:
1. ✅ **探索性分析** - 需要全面变量覆盖
2. ✅ **跨组比较** - 需要统一变量集
3. ✅ **方法学验证** - 需要最全面的框架
4. ✅ **复杂中介分析** - 需要完整中介变量

### 选择交互项方法的场景:
1. ✅ **计算效率优先** - 资源有限时
2. ✅ **假设验证** - 专注于核心变量
3. ✅ **调节效应分析** - 交互项设计明确
4. ✅ **初步筛查** - 快速识别强因果关系

## 关键发现

1. **标准化方法显著影响因果发现稀疏性**
   - 全局标准化: 更稀疏 (4.9%)
   - 交互项: 更密集 (12.7%)

2. **变量数量是关键驱动因素**
   - 更多变量 → 更稀疏的图
   - 更少变量 → 更密集的图

3. **两种方法互补而非竞争**
   - 全局标准化: 全面性、可比性
   - 交互项: 效率、专注性

## 建议工作流

1. **初步分析**: 使用交互项方法快速筛查
2. **深入验证**: 使用全局标准化方法全面分析
3. **结果整合**: 结合两种方法的发现
