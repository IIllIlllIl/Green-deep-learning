关键技术问题详解
================================================================================

问题1: 稳定边为什么这么少?
├─ 表现: group1_parallel有0条稳定边
├─ 根本原因: 3次运行在统计上无法产生显著稳定边
│           (二项式检验: 3/3的p=0.125 > 0.05)
├─ 解决方案: 增加至5-7次运行
├─ 预期效果: 应能检测到≥3条稳定边
└─ 优先级: P0 (立即)

问题2: 为什么与全局分析一致性这么差?
├─ 表现: group1_parallel只保留29%的全局边
├─ 可能原因:
│  1. 分层导致样本量减少 → 增加估计噪声
│  2. 真实的因果异质性 → 不同条件下因果关系不同
│  3. 分层标准不当 → 引入偏差
├─ 调查方法: 交互作用分析、异质性检验
└─ 优先级: P1 (本周)

问题3: ATE方向为什么有些相反?
├─ 表现: 3/12共同边的ATE符号不一致
├─ 可能原因:
│  1. 小样本波动 → ATE估计不稳定
│  2. 因果结构逆转 → 真实存在的交互作用
│  3. 计算错误 → 需验证
├─ 调查方法: bootstrap验证、符号分析
└─ 优先级: P1 (本周)

问题4: 多重比较校正为什么缺失?
├─ 影响: 29/33显著边中预期有~6条伪阳性(α=0.05)
├─ 建议: 应用FDR校正或Bonferroni校正
├─ 工作量: ~2小时
└─ 优先级: P2 (本周)
